## Getting Started
1. Clone the repo
   ```sh
   git clone https://github.com/gilgledson/vuejs-e-commerce
   ```
2. acesse fake api
   ```sh
   cd vuejs-e-commerce/api
   ```
3. install fake api
 ```sh
   npm install
 ```
4. run fake api
```sh
   npm start
 ```
5. Access front-end path
 ```sh
   cd vuejs-e-commerce/front-end
 ```
6. Build Docker image
  ```sh
   docker-compose up -d --build
 ```
7. access link
 ```sh 
    http://localhost:8081/
```
