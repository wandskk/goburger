## Getting Started
1. Clone the repo
 ```sh
   git clone https://github.com/wandskk/goburger.git
 ```
2. Go to project folder
 ```sh
   cd goburger
 ```
3. Install project requires
 ```sh
   npm install or npm i
 ```
4. Run React App
```sh
   npm start
 ```
5. Run json-server
```sh
   npm run backend
 ```
6. Access link
 ```sh 
    http://localhost:3000/
```
